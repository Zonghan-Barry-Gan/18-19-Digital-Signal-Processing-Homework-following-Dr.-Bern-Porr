The whole set of code is designed to realize real time IIR analysis of a ecg device we assembled using USB-DUX and BioSigma. A link to video showing how to use this device is uploaded to youtube https://youtu.be/1t3rUib9ENI

The I/O stream of USB-DUX is realized by applying code provided by Dr. Bern 
Bernd.Porr/pyusbdux, GitHub. (2019). berndporr/pyusbdux. [online] Available at: https://github.com/berndporr/pyusbdux [Accessed 5 Nov. 2019].)
